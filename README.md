**Tic-Tac-Toe** is a solved game, meaning when played optimally it will always end up in a tie.

Hence, this version of Tic-Tac-Toe allows you to make a normal move, or opt to roll dice against your opponent. If you win the dice, you can swap
any of the opponent's already placed pieces with one of yours. But if you lose, your turn would be skipped.
